# DigitalBank Backend

Application backend pour la gestion d'une banque digitale.

**Auteur :** Mustapha ELMIFDALI  
**GitHub :** [Mustapha-ELMIFDALI](https://github.com/Mustapha-ELMIFDALI)  
**Repository :** https://github.com/Mustapha-ELMIFDALI/DigitalBank-backend

## Technologies utilisées
- Java 17
- Spring Boot 3.2.5
- Spring Data JPA
- MySQL
- Lombok
- SpringDoc OpenAPI (Swagger)

## Lancement
```bash
mvn spring-boot:run
```

## API Documentation
Swagger UI disponible sur : http://localhost:8085/swagger-ui.html
